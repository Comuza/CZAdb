### Sobre el Proyecto

**Comuza-Optimizer** es una herramienta que utiliza **ADB (Android Debug Bridge)** para optimizar dispositivos móviles viejos o lentos.

- **Objetivo:** Revivir tablets/celulares sin conocimientos técnicos.
- **Filosofía:** Código limpio, sin bloatware, respeto por la privacidad del usuario.
- **Método:** Scripts automatizados vía ADB para deshabilitar servicios innecesarios, limpiar caché y optimizar el rendimiento.

### Características Planeadas

- Interfaz simple (posiblemente CLI o script batch)
- Deshabilitar apps del sistema que consumen recursos
- Eliminar aplicaciones
- Ajustes de animaciones y rendimiento

---

### Requisitos

- ADB instalado y configurado
- Depuración USB activada en el dispositivo
- Cable USB de Datos (o ADB inalámbrico)
